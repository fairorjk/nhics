# NHI Course Scheduler

교육기관의 교육과정 일정을 드래그 앤 드롭 간트 차트 및 월별 캘린더 뷰를 통해 손쉽게 관리하고 자동 편성하는 웹 기반 일정 관리 시스템입니다.

## 주요 기능

1. **교육과정 일정 자동 편성 제안**:
   - 등록된 교육과정의 희망 기한, 강의실 필요 여부, 정원 대기열 등을 반영하여 최적화된 연간 일정을 제안합니다.
2. **드래그 앤 드롭 간트 차트 (Gantt Chart)**:
   - 각 교육과정 블록을 마우스 드래그를 통해 직관적으로 일정을 조정할 수 있습니다.
3. **월별 캘린더 뷰 (Monthly Calendar)**:
   - 월간 전체 교육과정 운영 상황을 한눈에 파악할 수 있는 캘린더 뷰를 제공합니다.
4. **로컬스토리지 백업 및 파일 입출력**:
   - 데이터가 브라우저 로컬스토리지에 자동으로 보존되며, 전체 일정을 JSON 파일로 백업하고 복원할 수 있습니다.
5. **차트 고화질 이미지 내보내기**:
   - 현재 간트 차트와 캘린더 뷰를 깔끔하게 정렬된 PNG 고해상도 이미지 파일로 소장할 수 있습니다.
6. **라이트/다크 테마 지원**:
   - 시스템 테마 설정에 따라 쾌적한 다크 모드를 선택할 수 있습니다.

## 로컬 실행 방법

1. 본 저장소를 복사(Clone)합니다.
2. 터미널에서 다음 명령어를 실행하여 로컬 서버를 가동합니다.
   ```bash
   python server.py
   ```
3. 브라우저에서 `http://localhost:8000` 주소로 접속합니다.

## 배포 환경

본 애플리케이션은 순수 클라이언트 사이드 자바스크립트(Vanilla JS)와 CSS로 구동되므로, **Netlify**, **GitHub Pages**, **Vercel** 등 정적 웹 호스팅 환경에 별도의 빌드 과정 없이 즉시 배포할 수 있습니다.
